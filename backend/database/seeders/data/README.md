# Algeria Geography Seed Data

This directory contains the local seed dataset used by `Database\Seeders\AlgeriaGeographySeeder`.

Current files:

- `algeria_wilayas_58.json`
- `algeria_communes_1541.json`
- `LICENSE.kossa-algerian-cities.md`

Source:

- Repository: `kossa/algerian-cities`
- URL: https://github.com/kossa/algerian-cities
- Source files:
  - `database/seeders/json/Wilaya_Of_Algeria.json`

Important product note:

Law No. 26-06 of April 4, 2026 formalized a 69-wilaya territorial organization. This dataset intentionally keeps the 58-wilaya commune mapping with 1541 communes until an authoritative 69-wilaya commune reassignment dataset is integrated and tested.
